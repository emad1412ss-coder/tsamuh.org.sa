Humanity Charity Theme (RTL)
- Upload folder to wp-content/themes/humanity-charity
- Activate theme
- Create pages and assign templates:
  About -> About Page
  Projects -> Projects Page
  Donate -> Donate Page
  Reports -> Reports Page
  Volunteer -> Volunteer Page
  Contact -> Contact Page
- Set donate URL and contact info:
  Appearance -> Customize -> إعدادات الجمعية
